# Editing Guide (for academy staff)

You manage the website from the **Admin Dashboard** — no code needed.

## Signing in

1. Go to **`/admin`** on the website (e.g. `https://yoursite.com/admin`).
2. Enter your email and click **"Email me a sign-in link."**
3. Open the email and click the link — you're in. (No password to remember.)

> Only invited staff can sign in. Ask your developer to add your email in
> Supabase if your link doesn't work.

## What you can manage

Across the top tabs:

### Classes
- **Add / Edit / Delete** classes. Set the title, day, time, duration,
  instructor, room, **dance form** and **level**.
- The website's Schedule page and its filter dropdowns update automatically.
- Tick **"Featured workshop card"** to show the large highlighted card (then fill
  in the description and a banner image).

### Gallery
- **Add a photo** → choose **"Upload a photo"** and pick a file from your
  computer (or paste an image URL). Add a title, category, and caption.
- Edit or delete any photo. Changes show on the public Gallery page immediately.

### Instructors
- Add/edit/delete instructors, including **uploading their photo**.

### Enquiries
- See every enquiry sent from the Join page (name, email, message), and delete
  old ones. Click an email address to reply.

## Tips
- **Order** fields control the position (lower number = appears first).
- Changes are live the moment you save.
- Use **Sign out** (top right) when you're done on a shared computer.

## Things still in code (ask your developer)
- The logo/favicon (`public/images/logo.png`).
- Marketing text on the Home and About pages.
